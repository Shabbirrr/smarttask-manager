# 🚀 SmartTask Manager

A modern project management web application to organize tasks, track progress, and improve team productivity.

---

## 📌 Overview

SmartTask Manager is a web-based application designed to streamline project workflows. It allows users to create and manage projects, assign tasks, set deadlines, and monitor progress through an intuitive interface.

This project demonstrates practical implementation of modern frontend architecture and integration with real-world backend services.

---

## ✨ Features

* 🗂️ Multi-project and workspace support
* ✅ Task creation, assignment, and status tracking
* 📅 Deadline and scheduling management
* 📊 Progress tracking and basic analytics
* 👥 User authentication and access control
* ⚡ Responsive and user-friendly interface

---

## 🛠️ Tech Stack

* **Frontend:** ReactJS
* **State Management:** Redux Toolkit
* **Styling:** Tailwind CSS
* **Authentication:** Clerk
* **Database:** PostgreSQL (via Neon)
* **Workflows:** Inngest

---

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Shabbirrr/smarttask-manager.git
```

### Install dependencies

```bash
npm install
```

### Run the project

```bash
npm run dev
```
